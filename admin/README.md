## Setup

### For Development

- Run `npm i`
- Go to the [shared directory](../shared/)
  - Run `npm i`
  - Run `npm run build`
- Run `npm run link`
- Run `npm run start`

### To Install

- Run `npm i`
- Go to the [shared directory](../shared/)
  - Run `npm i`
  - Run `npm run build`
- Run `npm run link`
- Run `npm run dist`
- Go to the `dist` directory and install the relevant installer
